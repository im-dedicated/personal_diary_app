# Personal Diary App

A simple diary application maded that let users register, log in, create diary entries, edit them, delete them, and log out. All user and entry data is stored in JSON files under the data folder.

## Demo Mode

```bash
used render to deploy demo live link
```

Open https://personal-diary-app-xgy6.onrender.com for Live Demo version.

## Run locally

```bash
python app.py
```

Open http://127.0.0.1:8000 in your browser.

## Test

```bash
python -m unittest discover -s tests
```
